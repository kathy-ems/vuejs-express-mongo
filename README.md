## To start the app
1. Download the repository
`git clone https://github.com/kathy-ems/vuejs-express-mongo.git`

2. Install MongoDB shell version v4.2.17

3. Install dependencies
`npm install`

4. Start mongo
`mongod`

5. Start the server
`npm run dev`

6. Start the front end
`npm run serve`

7. Run the tests
`npm run test:unit`
